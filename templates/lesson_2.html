Урок № 2: "Простые типы данных: числа"

Для того чтобы программировать на python, необходимо разобраться с типами данных этого языка программирования. В Python есть несколько стандартных типов данных:

  • Numbers (числа)
  • Strings (строки)
  • Lists (списки)
  • Dictionaries (словари)
  • Tuples (кортежи)
  • Sets (множества)
  • Boolean (логический тип данных)

В рамках данного курса мы разберём не все, что показано на схеме. Для начала изучим самое основное, помеченное курсивом в списке. Давайте разберёмся поподробнее с каждым типом данных по отдельности.
Числа
Начнём с чисел - в python они бывают целыми (5, 12384, 0, -345), а также вещественными (11.25, 3.4925, -0.49). Целочисленные значения обозначаются int, а вещественные - float.
Целые числа (int)
Целые числа в python ничем не отличаются от обычных чисел. Они поддерживают набор самых обычных математических операций:

a = 3
b = 2

  • сложение

    print(a + b) --> 5

  • вычитание

    print(a - b) --> 1

  • умножение

    print(a * b) --> 6

  • целочисленное деление (вычисление целого частного)

    print(a // b) --> 1

  • целочисленное деление (вычисление целого остатка от деления)

    print(a % b) --> 1

  • деление (РЕЗУЛЬТАТ ДЕЛЕНИЯ - ТИП FLOAT)

    print(a / b) --> 1.5

  • возведение в степень

    print(a ** b) --> 9

Вещественные числа (float)
float - числа с плавающей точкой. Как именно они устроены - знать на данном этапе необязательно. Важно знать, что они неточные Как это понимать? Сейчас приведём пример. При сложении чисел 0.7 и 0.2 python вместо ожидаемого ответа 0.9 выдаёт число 0.8999999999999999 Можете проверить это самостоятельно, скопировав в свою программу команду print(0.7 + 0.2). Операции с вещественными числами идентичны операциям с целыми числами.
Но как можно перевести число из одного типа данных в другой? Для этого есть команды int() и float(). В скобки записываем необходимое число или переменную.

print(int(2.345))

>>> 2

После изучения этих двух типов данных возникает вопрос: Как задать определённое значение переменной? Сейчас мы вместе напишем простую программу на python.
Присваивание. Первая программа на python

Задание:

Дан прямоугольник со сторонами a и b. Необходимо найти площадь прямоугольника и вывести её. Значения a и b пользователь вводит с клавиатуры в соответствующем порядке; каждое значение на новой строке.

Первая программа будет состоять всего из четырёх строк. Первые две строки - получение данных от пользователя с помощью команды input(), которую мы изучили на первом уроке. Не забываем, чтобы получить именно целочисленное значение и присвоить его переменной, необходимо не забыть перевести то, что мы получаем от пользователя в целое число функцией int(). Чтобы сократить запись сделаем так: a = int(input()). Знак "=" присваивает переменной a значение, введённое пользователем, переведённое в целочисленный тип данных. Далее делаем точно так же со второй переменной b. Следующий шаг - вычисление площади. Из математики нам известно, что площадь прямоугольника - это произведение его сторон. Таким образом, мы создаём переменную s и присваеваем ей значение произведения переменных a и b. Это третья строка нашего кода. В четвёртой строке мы просто выводим переменную c уже знакомой нам командой print(). Вот и программа готова.

a = int(input())
b = int(input())
s = a * b
print(s)
