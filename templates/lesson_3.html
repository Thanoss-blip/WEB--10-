Урок № 3: "Простые типы данных: строки"

Итак, мы с вами уже рассмотрели числовые типы данных: int и float. Далее будем рассматривать ещё один протой, но очень важный тип данных - строки. В python они называются str. Они похожи на те строки, которые ученики обычно пишут в тетрадях. Только в python строки записываются в ковычках - одинарных или двойных.

"Привет, мир!" # строка

Однако для удобства работы символы в строках пронумерованы, или проиндексированны. Но считать мы начинаем не с единицы, как учили в начальной школе. Счёт в python ведётся с нуля. Таким образом, все символы в строке слева направо пронумерованы, начиная с нуля.

Кроме того, существует индексация справа налево. Такая индексация начиается с конца слова с индекса -1. Далее каждый индекс последовательно уменьшается на одну единицу

Зная индекс каждого символа, можно получить значение (данный символ), зная его индекс. Допустим, нам необходимо получить значение пятого символа некой строки a и записать это значение в переменную b. Это мы можем сделать так: название_строки[индекс]. Помните, что пятый по счёту символ будет иметь индекс 4, потому что отсчёт начинается С НУЛЯ.

a = "Привет, мир!"
b = a[4]
print(b)

>>> е

Но в каких случаях может пригодиться другой способ индексации? (справа налево). Допустим, у нас есть длинная строка. Например, длина её больше 100 символов. Тогда удобнее получить значение символа, находящегося ближе к концу строки будет удобнее, если обратиться к данному символу через отрицательный индекс. Пример:

a = "abcdefghijklmnopqrstuvwxyzabc..."
# далее идёт повтор строки 4 раза
b = a[100]
c = a[-4]
print(b, c)

>>> w w

Итак, на данный момент мы умеем получать только один символ строки по индексу. Но что же делать, если необходимо "извлечь" из строки последовательность символов? Это можно сделать с помощью срезов. Они работают так: после названия строки в квадратных скобках пишем область среза. Пример: a[3:6]. Эта запись означает, что осуществится срез строки с третьего по пятый символ включительно (речь именно об индексаз python). Так, стоп. Ничего не понятно. В скобках написано 3 и 6, а на деле получаем 3 и 5. Как так? Дело в том, что правая граница не включается в срез.

если не указать начальное значение среза, то он будет сделан от начала строки, если не указать конечное - до конца. Пример:

a = "Привет, мир!"
b = a[:4]
print(b)

>>> Прив

a = "Привет, мир!"
b = a[5:]
print(b)

>>> т, мир!

В срезах есть дополнительный параметр: шаг. Он указывается после правой границы через двоеточие и обозначает промежуток, через который буквы будут отбираться в срез. Примерно так:

a = "Привет, мир!"
b = a[3:10:2]
print(b)

>>> вт и

Шаг может быть и отрицательным, тогда буквы в срез будкт отбираться в обратном порядке, то есть слрава налево. Тогда нужно начинать с правой границы среза в слове, а заканчивать левой, так как в срез забираются буквы в обратном порядке.

a = "Привет, мир!"
print(a[10:3:-2])

>>> рм,е

Лайфхак
Чтобы просто перевернуть порядок символов в строке, достаточно сделать её срез не задавая значений границ среза (тогда строка будет учитываться целиком) и указать шаг -1, чтобы буквы из строки "отбирались" в срез без пропусков, а лишь в обратном порядке:

a = "Привет, мир!"
print(a[::-1])

>>> !рим ,тевирП

Кроме того, строки поддерживают ещё некоторые операции. Это конкатенация и умножение строки на число. Рассмотрим, как это работает. Кокатенация - сложение строк. То есть если нужно соединить несколько строк в одну, нам необходимо "скрепить" их знаком простого сложения, то есть знаком "+".

a = "Привет, "
b = "мир!"
print(a + b)

>>> Привет, мир!

Умножение строк тоже сужествует. Только умнжать строку на строку нельзя. Только строку на число. И тогда данная строка будет повторяться столько раз, на сколько мы её умножили. Проще говоря мы "прибавляем" к нужной строке её же саму столько раз, на сколько мы её умножили. Ничего не понятно? Сейчас покажем.

a = "Мама"
b = 5
print(a * b)

>>> МамаМамаМамаМамаМама
